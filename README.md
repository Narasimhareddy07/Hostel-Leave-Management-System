# 🏨 Hostel Leave Management System

A web-based application that allows students to apply for hostel leave online and enables administrators to manage, approve, or reject requests efficiently. This system replaces manual processes with a digital solution for better tracking and transparency.

---

## 🚀 Features

* 👨‍🎓 Student login and registration
* 📝 Apply for leave with reason and dates
* ⏳ Leave status tracking (Pending / Approved / Rejected)
* 👨‍💼 Admin dashboard to manage requests
* 🗄️ Stores data securely in database
* 📊 Organized and easy-to-use interface

---

## 🛠️ Technologies Used

* Python / PHP (based on your implementation)
* HTML, CSS, JavaScript
* MySQL
* Flask / Django (if used)

---

## ⚙️ How It Works

1. Student logs into the system
2. Submits a leave request with details
3. Request is stored in the database with **Pending** status
4. Admin reviews the request
5. Admin approves or rejects the leave
6. Student can view updated status
